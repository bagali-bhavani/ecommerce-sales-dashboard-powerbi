# 📊 E-Commerce Sales Dashboard | Excel & Power BI

## 📌 Project Overview

This project is an interactive **Business Intelligence dashboard** built using **Microsoft Excel** and **Power BI** to analyze e-commerce sales performance and support data-driven business decisions. The solution transforms raw sales data into meaningful insights through data modeling, DAX calculations, interactive visualizations, and time intelligence analysis.

The project follows a complete data analytics workflow, including data preparation, data modeling, relationship building, KPI development, dashboard design, and business insight generation. It demonstrates practical skills in **Excel, Power BI, Power Query, DAX, Data Modeling, Business Intelligence, and Data Visualization**, making it suitable for business reporting and performance analysis.

---

## 🎯 Business Problem

E-commerce businesses generate large volumes of transactional data every day. However, raw data alone does not provide meaningful insights for decision-making. Business stakeholders require an interactive reporting solution that can monitor sales performance, profitability, product performance, customer segments, shipping methods, and regional trends in real time.

This dashboard addresses these challenges by converting raw transactional data into an interactive and easy-to-understand Business Intelligence solution that supports faster and more informed business decisions.

---

## 🎯 Project Objectives

- Build an interactive Business Intelligence dashboard using Excel and Power BI.
- Analyze Year-to-Date (YTD) Sales, Profit, Quantity Sold, and Profit Margin.
- Compare current year performance with Previous Year-to-Date (PYTD).
- Track business performance using dynamic KPI cards.
- Identify the Top 5 and Bottom 5 performing products based on YTD Sales.
- Analyze sales performance across different product categories.
- Compare sales distribution by shipping type and customer segment.
- Visualize sales distribution across different U.S. states using map visuals.
- Implement Time Intelligence calculations using DAX.
- Design an interactive dashboard that enables users to filter and explore business performance efficiently.
